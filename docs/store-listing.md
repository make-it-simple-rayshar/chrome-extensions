## Full Description (EN)

```
Full Page Screenshot captures entire web pages — not just the visible area. Click once, and the extension scrolls the page, captures each section, and stitches them into a single high-resolution PNG.

✦ WHY THIS ONE?

Most screenshot extensions only capture the document body. Full Page Screenshot goes further:

• Scrollable containers — captures SPAs like Gmail, Slack, Notion, and admin dashboards where the main content scrolls inside a container, not the page itself.
• Element picker — click any element on the page to capture just that section. Scrollable elements get the full scroll-and-stitch treatment automatically.
• Cross-origin iframes — captures content inside embedded iframes (e.g., Twilio console, embedded forms).
• Pixel-perfect — respects your display's DPR (Retina-ready). No blurry screenshots.
• Zero data collection — everything stays on your device. No accounts, no cloud, no tracking. Open source.

✦ HOW IT WORKS

1. Click the extension icon
2. Choose "Capture Full Page" or "Select Element"
3. Screenshot saves automatically to your Downloads folder

The extension uses Chrome's captureVisibleTab API with a scroll-and-stitch technique. It scrolls the page (or a scrollable container), captures each viewport, and stitches them together using OffscreenCanvas — all in the background, without any visible glitches.

✦ WORKS ON

• Regular websites and blogs
• Single Page Applications (React, Angular, Vue)
• Admin dashboards with scrollable panels
• Pages with sticky headers and footers (automatically hidden during capture)
• Cross-origin iframe content

✦ PERMISSIONS

• activeTab — access current tab when you click the button
• scripting — scroll and measure the page
• downloads — save screenshots to your computer
• webNavigation — detect iframes for capture

No data leaves your browser. No analytics. No accounts. Just screenshots.

Full Page Screenshot is open source: https://github.com/ArekCzekala/chrome-extensions-rayshar
```
